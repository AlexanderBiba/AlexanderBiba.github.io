# Welcome to My Blog

This is an example blog post to demonstrate the blog functionality.

## Features

The blog supports:

- **Markdown** formatting
- Code blocks
- Images from `src/assets/blog-images/`
- Frontmatter for metadata

### Code Example

Here's a code example:

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```
